# Operadores
Javascript sendo uma linguagem C like, possue os mesmos operadoress.
- `=` - atribuição
- `-` - subtração
- `*` - multiplicação
- `/` - divisão
- `%` - módulo
- `+=` - atribuição com adição
- `*=` - atribuição com multiplicação
- `/=` - atribuição com divisão
- `%=` - atribuição com módulo
